NTPU_MIS_EN
=====================

This project is built by [react-bootstrap](http://react-bootstrap.github.io)

## key setting
1. coffee script
2. reactjs
3. browserify
4. compass

# Install

```shell
npm i && bower i && bundle install
```

# How to run it?

```shell
gulp server
```
and open browser <http://localhost:3000>

# Test

```shell
gulp test
```

# Build project

```shell
gulp build
```

# Build project as production

```shell
NODE_ENV=production gulp
```
